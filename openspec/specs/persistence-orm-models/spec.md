## Data Schema

### installation_accounts

| Column | Type | Nullable | Default | Notes |
|--------|------|----------|---------|-------|
| id | UUID | NO | gen_random_uuid() | PK |
| installation_id | VARCHAR | NO | — | GitHub installation ID, unique |
| account_login | VARCHAR | NO | — | GitHub account/org name |
| account_type | VARCHAR | NO | — | "User" or "Organization" |
| created_at | TIMESTAMP WITH TIME ZONE | NO | now() | |
| updated_at | TIMESTAMP WITH TIME ZONE | NO | now() | |

Indexes:
- `ix_installation_accounts_installation_id` UNIQUE on `installation_id`

### translation_tasks

| Column | Type | Nullable | Default | Notes |
|--------|------|----------|---------|-------|
| id | UUID | NO | gen_random_uuid() | PK |
| task_id | VARCHAR | NO | — | Domain task ID, unique |
| installation_id | VARCHAR | NO | — | FK-like reference to installation |
| repository | VARCHAR | NO | — | owner/repo full name |
| base_branch | VARCHAR | NO | — | |
| source_files | JSON | NO | — | List of file paths |
| language | VARCHAR | NO | — | Target language code |
| status | VARCHAR | NO | 'queued' | queued/running/succeeded/failed |
| pr_url | VARCHAR | YES | — | Set on success |
| pr_number | INTEGER | YES | — | Set on success |
| file_mappings | JSON | YES | — | List of {source_path, target_path} |
| error_code | VARCHAR | YES | — | Set on failure |
| error_message | VARCHAR | YES | — | Set on failure |
| created_at | TIMESTAMP WITH TIME ZONE | NO | now() | |
| updated_at | TIMESTAMP WITH TIME ZONE | NO | now() | |

Indexes:
- `ix_translation_tasks_task_id` UNIQUE on `task_id`
- `ix_translation_tasks_installation_id` on `installation_id`
- `ix_translation_tasks_created_at` on `created_at` DESC (recent task lookup)

### translation_files

| Column | Type | Nullable | Default | Notes |
|--------|------|----------|---------|-------|
| id | UUID | NO | gen_random_uuid() | PK |
| task_id | VARCHAR | NO | — | References translation_tasks.task_id |
| source_path | VARCHAR | NO | — | Original file path |
| target_path | VARCHAR | NO | — | Translated file path |
| status | VARCHAR | NO | 'pending' | pending/translated/failed |
| created_at | TIMESTAMP WITH TIME ZONE | NO | now() | |

Indexes:
- `ix_translation_files_task_id` on `task_id`

## ORM Model Rules

- All models use `Mapped` type annotations (SQLAlchemy 2.x style).
- No imports from `app/domain/`, `app/controller/`, `app/services/`.
- `source_files` and `file_mappings` stored as JSON columns.
- Status values constrained by application logic, not DB enum (allows future extension).
- UUID primary keys generated by database (`gen_random_uuid()`).
